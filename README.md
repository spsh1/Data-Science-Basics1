# DS-Fundamentals

## Overview

This repository presents a professional collection of introductory data science notebooks and visualization examples created with Python. It is intended to demonstrate core concepts in exploratory data analysis, charting, numerical computing, and animation using libraries such as NumPy, Pandas, and Matplotlib.

## Repository Contents

- `NumPy_Basics.ipynb` — foundational numerical computing with NumPy arrays and vectorized operations
- `Pandas.ipynb` — data manipulation, aggregation, and analysis using Pandas DataFrames
- `1. Matplotlib_Line_plots.ipynb` — line chart creation, styling, and presentation techniques
- `Barplot and Horizontal Bar Plot.ipynb` — bar chart design, comparison, and layout choices
- `Scatter Plot.ipynb` — scatter plot customization and data relationship exploration
- `Histogram and Boxplot.ipynb` — distribution analysis with histograms and boxplots
- `Violin plot, pie chart and Donut Chart.ipynb` — categorical visualization techniques for different chart types
- `Other_plots.ipynb` — additional Matplotlib examples and advanced plotting patterns
- `animations.py`, `circle_animations.py`, `wireframe_animations.py` — animated visual demonstrations for dynamic plotting

## Key Skills Demonstrated

- Data visualization using Matplotlib
- Statistical distribution and exploratory analysis
- Data preparation and transformation with Pandas
- Numerical problem solving with NumPy
- Building visual storytelling examples for analytics
- Creating Python-based animation scripts

## Getting Started

### Prerequisites

- Python 3.8 or later
- Jupyter Notebook, JupyterLab, or VS Code with the Python extension
- Python packages:
  - `numpy`
  - `pandas`
  - `matplotlib`

### Installation

Install the required packages with pip:

```bash
pip install numpy pandas matplotlib
```

If you use Jupyter locally, install it as well:

```bash
pip install notebook
```

## Usage

1. Open any notebook file (`.ipynb`) in Jupyter or VS Code.
2. Run the notebook cells sequentially to reproduce the examples.
3. Review the chart styling and code comments to understand each visualization.
4. Run animation scripts from the terminal when needed:

```bash
python animations.py
python circle_animations.py
python wireframe_animations.py
```

## Recommended Workflow

- Start with `NumPy_Basics.ipynb` to refresh array operations.
- Continue to `Pandas.ipynb` for data handling and analysis.
- Complete the Matplotlib notebooks to build visualization skills.
- Explore the animation scripts to see how plots can be made dynamic.
