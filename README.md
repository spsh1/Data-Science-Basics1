# DS-Fundamentals

## Overview

This repository presents a professional collection of introductory data science notebooks and visualization examples created with Python. It is intended to demonstrate core concepts in exploratory data analysis, charting, numerical computing, and animation using libraries such as NumPy, Pandas, and Matplotlib.

## Getting Started

### Prerequisites

- Python 3.8 or later
- Jupyter Notebook, JupyterLab, or VS Code with the Python extension
- Python packages:
  - `numpy`
  - `pandas`
  - `matplotlib`

### Installation

Install the required packages with pip:

```bash
pip install numpy pandas matplotlib
```

If you use Jupyter locally, install it as well:

```bash
pip install notebook
```

## Usage

1. Open any notebook file (`.ipynb`) in Jupyter or VS Code.
2. Run the notebook cells sequentially to reproduce the examples.
3. Review the chart styling and code comments to understand each visualization.
4. Run animation scripts from the terminal when needed:

```bash
python animations.py
python circle_animations.py
python wireframe_animations.py
```

## Recommended Workflow

- Start with `NumPy_Basics.ipynb` to refresh array operations.
- Continue to `Pandas.ipynb` for data handling and analysis.
- Complete the Matplotlib notebooks to build visualization skills.
- Explore the animation scripts to see how plots can be made dynamic.
