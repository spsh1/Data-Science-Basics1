# DS-Fundamentals

## Overview

This repository presents a professional collection of introductory data science notebooks and visualization examples created with Python. It is intended to demonstrate core concepts in exploratory data analysis, charting, numerical computing, and animation using libraries such as NumPy, Pandas, and Matplotlib.

---

## Repository Contents

### NumPy Basics
Introduction to numerical computing using NumPy, including array operations, indexing, broadcasting, and mathematical computations.

### Pandas
Covers data manipulation and analysis using Pandas, including DataFrames, Series, filtering, grouping, and basic data cleaning techniques.

### Matplotlib Line Plots
Demonstrates how to create and customize line plots for data visualization and trend analysis.

### Barplot and Horizontal Bar Plot
Explains categorical data visualization using bar charts and horizontal bar representations.

### Scatter Plot
Focuses on relationship visualization between variables using scatter plots.

### Histogram and Boxplot
Covers distribution analysis using histograms and boxplots for statistical insights.

### Violin Plot, Pie Chart and Donut Chart
Shows advanced visualization techniques for distribution and composition analysis.

### Other Plots
Includes additional visualization examples for extended learning and experimentation.

---

## Key Skills Demonstrated

- Data visualization using Matplotlib  
- Statistical distribution and exploratory data analysis  
- Data preprocessing and transformation using Pandas  
- Numerical computing using NumPy  
- Visual storytelling for analytics  
- Python-based animation scripting
